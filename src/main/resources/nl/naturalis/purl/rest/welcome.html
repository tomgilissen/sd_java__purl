PURL server up and running!

Welcome to the biodiversitydata.nl PURL server. We are in the process of creating persistent URLs (PURLs) for our main assets. For more information, see our documentation.

PURLs take the following form:

Naturalis:

@baseurl@/naturalis/specimen/{ObjectID}

Substitute {ObjectID} with an actual specimen ID. For example: @baseurl@/naturalis/specimen/U.1040749

Xeno-canto:

@baseurl@/xeno-canto/observation/{ObjectID}

Substitute {ObjectID} with an actual observation ID from xeno-canto.org. For example: @baseurl@/xeno-canto/observation/XC256131
